# FUTURE_FS_02
Mini CRM – Client Management System
Internship Task Submission – Future Interns

This project was developed as part of the Full Stack Web Development Internship at Future Interns.

The objective of this task was to build a functional and professional web-based CRM (Client Relationship Management) system using core web technologies.

Features Implemented
--------------------

Interactive Dashboard with Lead Analytics
Add / Edit / Delete Leads
Professional Centered Modals (View & Delete Confirmation)
Lead Status Tracking (New / Contacted / Converted)
Follow-up Date & Notes Management
Search & Filter Functionality
Sidebar Navigation with Active Page Highlight
Responsive & Clean UI Design
